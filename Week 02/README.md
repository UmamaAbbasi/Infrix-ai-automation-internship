# Week 02 - AI Powered Chatbot Using Scraped Data

## Project Overview

This project was developed as part of my AI & Automation Internship at Infrix.

The chatbot collects data from a website using web scraping techniques and allows users to search and interact with the collected information through a Streamlit web interface.

## Features

* Website Data Scraping
* Data Storage in Text File
* AI-Powered Search Chatbot
* Interactive Streamlit Interface
* User-Friendly Design

## Technologies Used

* Python
* Requests
* BeautifulSoup
* Streamlit

## Project Files

* web_scrapper.py
* app.py
* training_data.txt
* requirements.txt

## How to Run

Install dependencies:

```bash
pip install -r requirements.txt
```

Run scraper:

```bash
python web_scrapper.py
```

Run chatbot:

```bash
streamlit run app.py
```

## Outcome

Successfully scraped website data and built an AI-powered chatbot capable of retrieving relevant information from the collected dataset.
